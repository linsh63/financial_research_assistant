### DeepDOC

OCR 包含了两个模块：

* TextRecognizer(文字识别)：给定带文字图片，会识别出文字
* TextDetector(找到文字框)：把文字按区域划分，每个区域用4个点来标定

OCR 的使用示例以及返回的结果如下：

```python
ocr = OCR()
result = ocr(page_img)
# result: [(box, (text, score)), ...]
# box 是四点框坐标，text 是识别文本，score 是识别置信度。

only_boxes = list(ocr.detect(page_img))
text = ocr.recognize(page_img, only_boxes[0][0])
```

### chunk
